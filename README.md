# Quadratic Equations Solver

Solve roots of Quadratic Equations.

# How to Use
Import get_roots() from quadratic_equation.py and call it with 3 arguments(number, number, number). 


Example: 
<pre>
    <code>
    $ python3
    </code>
</pre>

<pre>
    <code>
    >>> from quadratic_equation import get_roots
    >>> get_roots(1,2,3)
    (None,None)
    </code>
</pre>
